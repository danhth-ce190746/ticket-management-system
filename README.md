# Ticket Management System

## Prerequisites

- Node.js v18+
- npm
- Git

## Setup Instructions

### Clone repository

git clone https://github.com/danhth-ce190746/ticket-management-system.git

### Install dependencies

npm install

### Run development server

npm run dev

### Open browser

http://localhost:3000
